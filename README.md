# ISO 27001 Risk Assessment

This beginner GRC project demonstrates a basic ISO 27001 risk assessment process for a fictional healthcare company called CloudCare Medical.

- Asset Identification
- Security Controls
- ISO 27001 Concepts
- Risk Management
- Documentation

- Asset Inventory
- Risk Assessment
- Security Controls
